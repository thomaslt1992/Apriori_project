In order to run this script you have to have python (ver 3.7 or higher) and jupyter notebook in your computer

All the csv files should be present in the same folder with the script in order to run properly.

You will need some packages in order to run this script as well:

More specifically,

Altair: https://altair-viz.github.io/
To download altair via pip: pip install altair
To download vega (mandatory) via pip: pip install vega
To download via anaconda: conda install -c conda-forge altair vega_datasets

// We highly suggest to download altair via conda! //

==========================================================================================================
Matlplotlib_venn

To download Venn diagrams via conda: conda install -c conda-forge matplotlib-venn
To download Venn diagrams via pip: pip install matplotlib-venn

==========================================================================================================
Mlxtend: http://rasbt.github.io/mlxtend/

To download MLxtend via conda: conda install -c conda-forge mlxtend 
To download MLxtend via pip: pip install mlxtend

==========================================================================================================
You will also need Pandas, Matplotlib and Sklearn-Scikit

Pandas: https://pandas.pydata.org/docs/
Matplotlib: https://matplotlib.org/
Sklearn: https://scikit-learn.org/stable/

To download them via pip:

Pandas: pip install pandas
Matplotlib: pip install matplotlib
Sklearn: pip install -U scikit-learn
